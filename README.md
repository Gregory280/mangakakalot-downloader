# mangakakalot-downloader
A ruby script to download all chapters from given manga in the webiste mangakakalot/manganelo. 

A ruby scrapper script to download all chapters from given manga in the Mangakakalot/Manganelo.

Copy and paste the manga's url into variable url and just run. It will create a folder with the manga's title and then create a specific folder for each chapter.

If you want to download just a specific chapter use the single chapter downloader. Just paste the chapter's url and run it.


Through your terminal run the following gem installation in order to run the scripts
`

gem install nokogiri
gem install rest-client
gem install open-uri
gem install fileutils

`
